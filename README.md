# GAERHA

### Significado
GAERHA viene de las siglas **G**estor de **A**sistencia de **E**mpleados, **R**ecuros **H**umanos y **A**dministradores.

### ¿Cómo instalar GAERHA?
En [Releases](https://github.com/DaniPraivet/GAERHA/releases) os teneis que instalar la última versión. 

Una vez instalado en tu equipo debeis hacer lo siguiente:
1. Tener instalado [Microsoft OpenJDK 21.0.10](https://learn.microsoft.com/es-es/java/openjdk/older-releases#openjdk-21)
2. Descomprimir la carpeta comprimida .ZIP
3. Ejecutar el archivo .bat
